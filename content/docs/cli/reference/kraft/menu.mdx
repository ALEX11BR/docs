---
title: "kraft menu"
description: Open's Unikraft configuration editor TUI
---

Open Unikraft's configuration editor TUI

```
kraft menu [DIR]
```

### Examples

```
# Open configuration editor in the cwd project
$ kraft menu

# Open configuration editor for a project at a path
$ kraft menu path/to/app
```

## Options

```
  -m, --arch string        Filter prepare based on a target's architecture
  -h, --help               help for menu
      --kraftfile string   Set an alternative path of the Kraftfile
  -p, --plat string        Filter prepare based on a target's platform
  -t, --target string      Filter prepare based on a specific target
```
