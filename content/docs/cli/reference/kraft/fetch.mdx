Fetch a Unikraft unikernel's dependencies

```
kraft fetch [DIR]
```

### Examples

```
# Fetch the cwd project
$ kraft fetch

# Fetch a project at a path
$ kraft fetch path/to/app
```

## Options

```
  -m, --arch string        Filter prepare based on a target's architecture
  -h, --help               help for fetch
      --kraftfile string   Set an alternative path of the Kraftfile
  -p, --plat string        Filter prepare based on a target's platform
  -t, --target string      Filter prepare based on a specific target
```
